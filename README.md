# JPEG

### "A photo is worth a thousand lines of code" 

JPEG was an epic 25 hour all-girls game jam that ran from July 5-6th, 2025 in Ottawa. 

This repository contains the source code for [our website](https://jpeg.hackclub.com/). It was made with HTML, TailwindCSS (styling and adding responsitivity), JavaScript (the cool animations) and late night meetings.
