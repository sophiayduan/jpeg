# JPEG

### "A photo is worth a thousand lines of code" 

JPEG was an epic 25 hour all-girls game jam that ran from July 5-6th, 2025 in Ottawa, Canada. 

This repository contains the source code for [our website](https://jpeg.hackclub.com/). 

It was made with HTML, TailwindCSS, JavaScript, and late night meetings.
